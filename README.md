Application web mobile dédiée au signalement d’animaux perdus, errants ou décédés sur l’ensemble du territoire français.

Cette application a pour objectif de centraliser les signalements au sein d’une plateforme unique, accessible à tous, 
afin de faciliter la diffusion des informations et d’améliorer la réactivité des acteurs concernés (propriétaires, citoyens, 
bénévoles, associations).
